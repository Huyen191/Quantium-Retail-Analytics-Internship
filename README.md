### Project Overview
Supported the **Chips Category Manager** in strategic planning by analyzing customer behavior and evaluating a **nationwide store layout trial** using real transaction data.

### Tasks Completed

| Task | Objective | Key Achievements |
|------|--------|------------------|
| **Task 1** | Customer Segmentation & Category Insights | • Cleaned & enriched transaction data<br>• Extracted brand & pack size with regex<br>• Identified top segments: Older Families (Budget), Retirees, Young Singles/Couples (Mainstream)<br>• Discovered Premium Young Singles/Couples pay 20% more per packet |
| **Task 2** | Store Layout Trial Evaluation (A/B Testing) | • Matched control stores using correlation + magnitude distance<br>• Measured sales uplift:<br>  Store 77: **+17.2%** (p < 0.01)<br>  Store 86: **+8.7%** (p < 0.05)<br>  Store 88: no significant change<br>• Performed statistical t-tests |
| **Task 3** | Executive Presentation | • Delivered client-ready recommendation to Category Manager (Julia)<br>• **Final verdict: YES – Roll out nationwide**<br>• Created 11-slide presentation using official Quantium template |

### Tech Stack
`Python` • `Pandas` • `Matplotlib` • `Seaborn` • `SciPy` • `Regex` • `Statistical Testing`


### Recommendation to Client
> **"YES – Roll out the new layout nationwide, prioritizing stores similar to 77 & 86. Expected category uplift: +9% to +12%."**
